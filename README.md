# Bewm
A 3d raycast engine im working on

# Requirements:
* SDL2

# Compiling:

## Mac/Linux:
```
sudo apt-get install cmake libSDL2-dev
```
or
```
brew install cmake
brew install sdl2
```
and then
```
mkdir build && cd build
cmake ..
make
```
