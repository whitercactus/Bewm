# Bewm
A 3d raycast engine im working on

# Compiling:

## Windows:
```
cmake -G "Visual Studio 14 Win64" src
```
### Open Visual Studio, and run the follwing in Visual Studio cmd
```
msbuild Bewm.sln
```

## Mac/Linux:
```
mkdir build && cd build
cmake ..
make
```
