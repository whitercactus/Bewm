# Bewm
A 3d raycast engine im working on

# Compiling:

## Windows:
```
cmake -G "Visual Studio 14 Win64" .
```
### Open Visual Studio, and run the follwing in Visual Studio cmd
```
msbuild Bewm.sln
```

## Mac/Linux:
```
sudo apt-get install cmake libSDL2-dev
```
or
```
brew install cmake
brew install sdl2
```
and then
```
mkdir build && cd build
cmake ..
make
```
